# Dashboard Visualization Skill
## Mission
Hiển thị scan analytics và vulnerability trends rõ ràng, hiện đại và dễ hiểu.
## Required Charts
- Risk distribution
- Severity distribution
- Scan trends
- Language distribution
## Visualization Rules
- Không overuse charts
- Ưu tiên readability
- Color severity consistent toàn app
## Severity Colors
- LOW → green
- MEDIUM → yellow
- HIGH → orange
- CRITICAL → red
## Dashboard Widgets
- Total scans
- Vulnerable scans
- Average confidence
- Most scanned language
## Chart Rules
- Responsive
- Tooltip rõ ràng
- Empty states đầy đủ
- Skeleton loading
## UX Rules
- Quan trọng nhất đặt trên cùng
- Không nhồi quá nhiều metrics
- Mobile stack layout
## Forbidden
- Không 3D charts
- Không rainbow colors
- Không tiny unreadable charts
## QA Checklist
- Charts responsive
- Colors consistent
- Data mapping đúng
- Empty/loading state đúng
